## 현재의 문제점
   * 사설 토토나 도박은 먹튀가 너무많음A
   * 국내의 경우 불법적이다 보니 제한도 없다. 
   * 운영자의 조작 가능성도 있음

## 해결방안은?
* 운영자가 손을 댈 수 없는 완전한 **탈 중앙화** 된 플랫폼을 이용하면된다.
* 그리고, 모든 배당률이나 조작의 논란이 있는부분을 100% 공개된 코드 내에서 실행한다
